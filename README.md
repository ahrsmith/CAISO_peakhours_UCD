2024 ACC tool was download here: https://www.cpuc.ca.gov/dercosteffectiveness

Tool was reformatted to consider 2019-2025 and handle new data: https://docs.google.com/spreadsheets/d/1ssNnrD8bd5MqvRy0U5z9GEQGXBiPIqUO/edit?usp=sharing&ouid=115005910739869519828&rtpof=true&sd=true
* yellow cells indicate original manual inputs by E3
* green cells indicate cells which are populated by historic data

new_inputs overwrites the template file with the new data. Results are read from the "Detailed Output" sheet. Results need to be calculated separately for each climate zone and IOU.

Processed data contains reformatted emissions and energy data.
Raw data contains data which needs minimal reformatting (transposition, etc.).
The PCAFs folder contains the generation (one file representing all of CAISO), distribution (one file for each IOU), and distribution (one file for each CZ) PCAFs. They are generated using the methodology described in Section 2 of the report.
